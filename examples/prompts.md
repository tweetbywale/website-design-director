# Prompt Examples

## Website Strategy Brief

```text
Use $website-design-director to create a website strategy brief for a new AI automation agency.

Include:
- audience
- positioning
- primary conversion goal
- homepage section order
- visual direction
- content hierarchy
- mobile-first notes
- trust signals
```

## UI/UX Audit Report

```text
Use $website-design-director to audit this website and produce a UI/UX report sheet.

Rate:
- visual hierarchy
- spacing
- typography
- responsiveness
- mobile UI
- conversion clarity
- animations
- accessibility
- overall polish
```

## Landing Page Improvement Plan

```text
Use $website-design-director to improve this landing page concept.

Tell me:
- what the hero should say
- what sections are missing
- what order the page should use
- what proof is needed
- what CTAs should appear
- how the mobile version should behave
```

## Brand Website Critique

```text
Use $website-design-director to critique this brand website as a creative director.

Focus on:
- brand positioning
- premium feel
- trust
- visual system
- storytelling
- conversion clarity
```
