# Security

## Public Repository Status

This repository is intended for public distribution. It should not contain secrets, private credentials, environment files, local machine paths, or private project data.

## Reporting A Concern

If you find a security or privacy issue in this repository, open a GitHub issue with a clear description of the problem and the affected file.

Do not include live secrets, tokens, passwords, or private credentials in public issues.

## Maintainer Checklist

Before publishing updates, check for:

- API keys or tokens.
- Passwords or credentials.
- `.env` files.
- Local machine paths.
- Private client or project data.
- Hidden workflow files that expose secrets.
- Generated build folders or dependency folders.
