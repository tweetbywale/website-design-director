# Website Review Example

Use this prompt when auditing a live website, concept, screenshot, or existing homepage.

```text
Use $website-design-director to critique this website:
https://example.com

Evaluate:
- strategic clarity
- audience fit
- visual hierarchy
- layout quality
- typography
- color system
- spacing
- mobile responsiveness
- conversion flow
- content clarity
- brand alignment
- trust signals
- interaction quality
- accessibility basics
- overall polish

Give me:
- scores out of 10
- top 3 strengths
- top 3 weaknesses
- priority fixes
- quick wins
- deeper redesign opportunities
- final recommendation
```

## Short Version

```text
Use $website-design-director to review this homepage and tell me what to fix before launch.
```
